This Streamlit dashboard displays emissions across industries and sectors.  
It includes:

- Interactive filters (Year, Sector)
- Emissions bar chart (Altair visualization)
- Chat panel for dataset questions
- Web lookup using Wikipedia API
- Downloadable CSV output

## How to Run Locally
1. Install dependencies  
pip install -r requirements.txt

2. Run Streamlit app  
streamlit run app.py

## Notes
- `sample_data.csv` contains example emissions data.
- To use real emissions data, replace this file with actual data.
- The dashboard is fully deployable on Streamlit Cloud.
